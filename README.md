TiebaManager
=========

[百度贴吧](https://tieba.baidu.com/)吧务管理工具  

* 自动扫描帖子并处理违规帖
* 违规规则支持关键词、正则表达式、图片（包括头像和签名）、等级、时间
* 支持插件开发
* 通过GPLv2协议开源
